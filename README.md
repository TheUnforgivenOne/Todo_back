# Todos app
Simple Todos app

## Features
1. Authentification
2. REST API
3. Functional UI
4. Attempt to build backend architecture

## Stack
1. Express
2. MongoDB
3. React

## Client
1. ```cd client```
2. ```npm install```

## Server
1. ```npm install```
2. ```npm run initial```
3. ```npm run server```

Fill DB_CONNECTION string inside .env.local

Run both client and server ```npm run dev```
